## Contexto

Sistema POS desarrollado para gestionar ventas, productos y operaciones básicas de un entorno comercial.

## Problema

Muchos negocios pequeños necesitan controlar ventas, productos y registros sin depender de herramientas excesivamente complejas o costosas.

## Solución

Se desarrolló una aplicación con backend, base de datos relacional y despliegue online en entorno cloud.

## Stack técnico

- Java
- Python
- PostgreSQL
- HTML/CSS
- Oracle Cloud

## Aprendizajes

Este proyecto me permitió trabajar con persistencia de datos, despliegue en servidor, configuración de entorno y estructura de aplicación real.

## Mejoras futuras

- Autenticación robusta.
- Dashboard de ventas.
- Reportes exportables.
- Mejor control de roles.
- Pruebas automatizadas.
