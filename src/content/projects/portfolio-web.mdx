## Contexto

Este portafolio fue diseñado como una pieza técnica para CV, no solo como una página de presentación.

## Problema

Un portafolio simple muestra información, pero no necesariamente evidencia criterio técnico, arquitectura o buenas prácticas.

## Solución

Se construyó una web rápida, modular y mantenible usando Astro, TypeScript, Tailwind CSS y Content Collections.

## Decisiones técnicas

- Astro para rendimiento y generación estática.
- TypeScript para robustez.
- MDX para agregar proyectos fácilmente.
- Componentes separados por responsabilidad.
- Estructura preparada para escalar.
